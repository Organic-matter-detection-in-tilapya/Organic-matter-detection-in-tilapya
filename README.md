🎯 Purpose

The main goal of this project is to:

Monitor organic matter levels in pond water

Help prevent water contamination

Improve tilapia health and survival rate

Provide real-time monitoring and alerts

Too much organic matter (fish waste, excess feeds, decaying plants) can reduce oxygen levels and harm the fish. This system helps detect that early.

🛠️ Features

📊 Real-time water condition monitoring

⚠️ Alert system for high organic levels

📈 Data visualization (graphs or dashboard)

💾 Data recording and storage

🖥️ User-friendly interface

⚙️ Technologies Used

Node.js

Electron (for desktop application)

JavaScript

HTML & CSS
