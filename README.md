🎯 Project Purpose

The main objective of this system is to monitor the organic matter level in tilapia pond water using a web-based simulation system developed in PHP.

Organic matter such as fish waste, excess feeds, and decaying plants can accumulate in the pond.
This can cause low dissolved oxygen, poor water quality, and may lead to fish stress or death.

This system helps users detect high organic levels early, allowing them to take preventive actions to maintain a healthy pond environment. 🐟💧

🛠️ System Features

📊 Simulated Real-Time Monitoring
Displays generated water condition data such as organic matter level.

⚠️ Alert Notification System
Provides warning messages when organic matter reaches critical levels.

📈 Data Visualization Dashboard
Shows monitoring results using charts and graphical reports.

💾 Data Recording and Storage
Stores simulated readings in a database for monitoring history and analysis.

🖥️ User-Friendly Web Interface
Simple dashboard design for Admin, Manager, and Staff users.

⚙️ Technologies Used

🐘 PHP – Handles system logic and server-side processing
🗄️ MySQL – Stores monitoring data and user information
🌐 HTML – Provides system structure and layout
🎨 CSS – Enhances system appearance and usability
📜 JavaScript – Enables simulation and dynamic dashboard updates

🧪 System Type

This project is a simulation-based monitoring system.
The data shown in the dashboard is system-generated and not from actual sensors.
It is intended for academic demonstration and learning purposes only.
